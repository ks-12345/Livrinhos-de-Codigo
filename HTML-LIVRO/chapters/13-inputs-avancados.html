13 - Inputs Avançados

Além dos tipos básicos, HTML5 trouxe novos tipos de input como date, color, number, file, range, url e tel.

Código

<form>
  Data: <input type="date" name="data"><br><br>
  Cor favorita: <input type="color" name="cor"><br><br>
  Idade: <input type="number" name="idade" min="0" max="120"><br><br>
  Arquivo: <input type="file" name="arquivo"><br><br>
  Volume: <input type="range" name="volume" min="0" max="100"><br><br>
  URL: <input type="url" name="site" placeholder="https://"><br><br>
  Telefone: <input type="tel" name="telefone" placeholder="(11) 99999-9999"><br><br>
  <button type="submit">Enviar</button>
</form>
