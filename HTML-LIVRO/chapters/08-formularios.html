08 - Formulários

Formulários permitem coletar dados do usuário. Eles incluem inputs de vários tipos, radio buttons, checkboxes, select, textarea e botões.

Código

<form action="/enviar" method="post">
  Nome: <input type="text" name="nome" placeholder="Digite seu nome" required><br><br>
  Email: <input type="email" name="email" required><br><br>
  Senha: <input type="password" name="senha" minlength="6" required><br><br>

  Gênero:
  <input type="radio" name="genero" value="M"> Masculino
  <input type="radio" name="genero" value="F"> Feminino<br><br>

  Interesses:
  <input type="checkbox" name="interesses" value="html"> HTML
  <input type="checkbox" name="interesses" value="css"> CSS
  <input type="checkbox" name="interesses" value="js"> JavaScript<br><br>

  Estado:
  <select name="estado">
    <option value="SP">São Paulo</option>
    <option value="RJ">Rio de Janeiro</option>
    <option value="MG">Minas Gerais</option>
  </select><br><br>

  Mensagem:<br>
  <textarea name="mensagem" rows="4" cols="40"></textarea><br><br>

  <button type="submit">Enviar</button>
  <button type="reset">Limpar</button>
</form>
