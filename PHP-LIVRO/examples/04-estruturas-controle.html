Estruturas de Controle

Exemplo usando if, else e switch.

Código PHP

<?php
$cor = 'azul';
if($cor == 'vermelho'){
 echo 'Parar';
}else{
 echo 'Continuar';
}
switch($cor){
 case 'azul': echo 'Olhe para o céu'; break;
 case 'verde': echo 'Olhe para a grama'; break;
}
?>
