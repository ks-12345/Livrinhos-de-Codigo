01 — Introdução ao Bootstrap

Breve: Bootstrap é um framework CSS/JS que facilita criar layouts responsivos. Abaixo a estrutura mínima e explicações linha-a-linha.

Estrutura mínima

<!-- 1: Tag do documento -->
<!doctype html>
<html lang="pt-BR">

<!-- 2: metadados obrigatórios para responsividade -->
<meta name="viewport" content="width=device-width, initial-scale=1">

<!-- 3: link para o CSS do Bootstrap (CDN ou local) -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- 4: script do bootstrap bundle antes do fechamento do body -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>

Comentários: container centraliza, row agrupa colunas e col-* define largura (soma 12).
